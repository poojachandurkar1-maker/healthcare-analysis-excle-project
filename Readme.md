# Health Outcome Analysis and Risk Stratification

## 📌 Project Overview
This project focuses on analyzing healthcare data using Microsoft Excel to identify patient health outcomes and perform risk stratification. The dataset includes patient demographics, laboratory reports, diagnoses, and outcome information to support healthcare decision-making.

The project demonstrates practical skills in:
- Data Cleaning & Preparation
- Healthcare Data Analysis
- Excel Functions & Formulas
- Dashboard Creation
- Data Visualization
- Risk Stratification

---

# 📂 Dataset Information

The Excel workbook contains multiple sheets:

## 1. Patients
Contains patient demographic information such as:
- Patient ID
- Age
- Gender
- Medical History

## 2. Labs
Includes laboratory test results:
- Blood Pressure
- Cholesterol Levels
- Glucose Levels
- Other Health Indicators

## 3. Diagnoses
Contains diagnosis-related details:
- Disease Type
- Diagnosis Codes
- Medical Conditions

## 4. Outcomes
Stores patient outcome data:
- Recovery Status
- Risk Category
- Readmission Indicators

## 5. Analysis
Contains:
- Pivot Tables
- KPI Analysis
- Risk Calculations
- Charts & Dashboard Reports
